## What is Mosy Script Site?

Mosy Script Site is my personal Roblox executor script hub. The tagline is *"Your one stop destination for all scripts."* It is a static website that lists available scripts, links to executors, and has a dedicated scripts page. The site was originally hosted publicly and used by the Roblox scripting community.

---

## Features

- **Script listing** — categorized list of available scripts
- **Scripts page** — dedicated page with script details and loader info
- **Clean dark UI** — minimal design focused on readability
- **Mobile responsive** — works on phones and tablets
- **Fast loading** — pure HTML/CSS/JS, no heavy frameworks

---

## How to Use

```bash
start index.html      # Windows
open index.html       # macOS
```

Pages: `index.html` (home) · `scripts.html` (script listings)
